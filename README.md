# video_freefuck

白嫖影视会员简易播放网站，基于flask，数据采集来自于最大资源网

## 简介

提交搜索框的影视片名，利用爬虫去搜索对应的最大资源网的对应的影片的片名，获取详细的每一集的播放地址（.m3u8的地址）, 然后通过hls.js的插件播放对应的m3u8的视频流。整个后端主要提供一个爬虫，采集到对应的视频地址后，返回给hls 进行播放。从此追番看剧，就不怕没会员了！！！
